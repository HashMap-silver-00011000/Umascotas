# Umascotas

Umascotas es una aplicación web que busca facilitar la **adopción responsable de animales callejeros o desamparados**.  
La idea principal es tener un sistema donde las personas puedan ver los animales disponibles, crear su perfil como adoptantes y hacer todo el proceso de adopción de forma más organizada.

---

## Tecnologías utilizadas

Por ahora el proyecto está hecho con:

- **Java Spring Boot** para el backend.  
- **MySQL** como base de datos relacional.  
- **TailwindCSS** y **JavaScript** para la parte visual (frontend aún en desarrollo).  

Mi idea es tener un diseño sencillo, pero funcional, que se vea bien en cualquier dispositivo.
